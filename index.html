What is Flask, And Why Do I Need it?

Flask Is a Python Micro Framework within python that allows developers to map information/requests being recieved from the server side of a webpage and validate it to-and-from the database to allow for easy comminication between fullstack applications. The Object Relational Mapper is made up of Models That are used to insntantiate objects of certain classes, ie; users. These models have Schemas which are used to check if the data coming in from the server side fit the data format needed to be sent to the database.


Think of Schemas Like the TSA, Or like a bouncer at a club checking to see if you are over 21.
